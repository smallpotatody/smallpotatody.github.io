对于WebStorm的叙述就到这儿了，“工欲善其事必先利其器”，下面来看看如何安装与破解WebStorm。

2. WebStorm的下载

去官网下载

WebStorm官网

点击DOWNLOAD进行下载

下载完成后双击打开进行安装

点击Next

选择自己的安装路径

根据自己电脑的位数选择，和添加到环境变量，.js .css .html勾选后之后js css html文件默认会用webstorm打开,然后点击Next。

点击Install

完成安装，选择I want to manually reboot later，然后点击Finsh。

3. 破解
